## Quick Start

### Prerequisites

- Node.js >= 16
- npm >= 8

### Setup

```sh
git clone <repository-url>
cd ipac-relocation
npm install
```

### Development

```sh
# Start local server
npm run dev

# Build for production
npm run build

# Preview build
npm run preview
```
